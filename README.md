### 第1刷、第2刷をお持ちの方

#### p.6 図1-1

**誤**

```
(5 - 1) * ((7 + 1) / (3 - 1))
      ↓
      4 * ((7 + 1) / (3 - 1))
      ↓
      4 * ( 8 ) / (3 - 1))
      ↓
      4 * ( 8 ) / ( 2 )
      ↓
      4 * 4.0
      ↓
      16.0
```

**正**

```
(5 - 1) * ((7 + 1) / (3 - 1))
      ↓
      4 * ((7 + 1) / (3 - 1))
      ↓
      4 * (8 / (3 - 1))
      ↓
      4 * (8 / 2)
      ↓
      4 * 4.0
      ↓
      16.0
```
